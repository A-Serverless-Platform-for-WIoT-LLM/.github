# 웨어러블 사물인터넷과 LLM의 실시간 통합을 위한 서버리스 아키텍처 기반 플랫폼 설계 및 구현: 노인 자기주도 케어 사례 중심으로
### A Serverless Platform for Real-Time Integration of Wearable IoT and Large Language Models: Design, Implementation, and an Elderly Self-Care Case Study

# Abstract
효과적인 노인 헬스케어는 자신의 의지와 주변의 개입을 통해 행동 변화를 촉진하는 것이다. 기존의 노인 헬스케어 서비스는 습관 개선을 위한 동기부여와 사회적 고립감 해소에 있어서 충분한 해결책을 가지고 있지 않다. 웨어러블 기기의 데이터를 인식하는 것에서 그친다면 실질적인 건강 개선 효과가 부족하다. 고립감은 노인 질환 및 사망률을 높이며, 독거노인의 경우 가족이 있는 노인에 비해 더 취약한 환경에 놓일 수 있다. 본 연구는 웨어러블 IoT 기술의 실시간 데이터 수집 능력과 LLM의 추론 능력을 결합하여, 실시간으로 개인 맞춤형 건강 분석을 제공하는 플랫폼을 구축하였다. Fitbit 데이터와 건강 문답 내역을 바탕으로 한 조언과 알림을 제공함으로써, 건강한 습관 형성에 적극적인 개입을 시도한다. 노인의 건강 분석 내용은 보호자와 의사에게도 공유되며, 조언 등록을 통해 간접적인 개입을 유도한다. 서버리스 아키텍처 기반 플랫폼으로서 확장성이 보장되며, 사용성 테스트에서 사용자의 긍정적 사용 경험을 얻은 바 있다.
